We are starting with Protarctor

Protractor is the node framework which we use with webdriver to run end to end test on web browser for angular js.

  1. Install node js from the parent website.
  2. run npm --version.
    If installed correctly, it will display the current installed version of node.js on your machine.
  3. run npm install -g protractor
    This installs the protractor on your system globally for all users.
  4. run protractor --version
    This will display the current version of the protractor installed on your system if installation is done correctly.
  5. run npm -g list
    This will display list of all nodes installed in your system for node.js
